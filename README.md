# ruvicorn-asgi
**Now Under Develop** 
ASGI Server implementation written in Rust.
Support HTTP/1.1 and Websockets.

## Plans
- HTTP/1.1 support
- Websocket support
- Execute as function call
- Execute as CLI
- Reloading in development mode.

## Features
- Fast.
- 100% python asyncio compatable
- Simple usage with minimal configuration
